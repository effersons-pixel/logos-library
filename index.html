Why Google Drive embeds fail — and the fix

Root cause 1 — Google Workspace domain restriction: If your file is in a work/school account, embedding is blocked even with "Anyone with the link" set.

Root cause 2 — X-Frame-Options: Google Drive sends SAMEORIGIN headers on many files, blocking iframe embedding in third-party sites.

Root cause 3 — File too large (>25 MB): Google's inline renderer won't process large PDFs.

✅ The fix: Use a personal gmail.com account → upload → set "Anyone with the link" → copy link. OR use Cloudinary (free tier, recommended).

Better alternative: Upload to archive.org for free hosting with no embed restrictions.
